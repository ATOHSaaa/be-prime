Amazonのセールで「<strong>20%OFF</strong>」と表示されていても、**過去にさらに安かった時期がある**ことは珍しくありません。買う前に価格の推移を確認したい人向けに、**[Keepa](https://keepa.com/#!language/5)**（キーパ）というAmazon価格トラッカーの使い方について解説します。Be Primeではセール情報を速報していますが、**最終的な買い時判断はAmazon.co.jpの表示と、必要ならKeepaのグラフをあわせて確認する**のがおすすめです。

## Keepaとは

[Keepa](https://keepa.com/)は、Amazon商品の**価格推移をグラフで可視化**するツールです。ブラウザ拡張機能を入れると、商品ページに過去の値動きが表示され、セール前後の価格を一目で比較できます。

| 項目 | 内容 |
|------|------|
| 正式名称 | Keepa - Amazon Price Tracker |
| 主な用途 | 価格推移の確認、値下げ通知（トラッキング） |
| 対応マーケット | Amazon.co.jp を含む各国のAmazon |
| 利用形態 | 公式サイト、Chrome拡張機能、スマホアプリ |
| 料金 | 基本機能は無料。詳細分析は有料プランあり |
| 公式サイト | [keepa.com（日本語表示可）](https://keepa.com/#!language/5) |

Keepaは物販・せどり向けの分析機能も充実していますが、**一般の買い物客でも「本当に安いか」を確認する道具**として十分使えます。

## こんな人におすすめ

| 向いている人 | 向いていない人 |
|-------------|---------------|
| セール価格が過去最安か知りたい | その場の表示価格だけで十分 |
| Prime Day・タイムセール前に候補をリスト化したい | 1回きりの低額日用品だけ買う |
| 欲しい商品の値下げを通知で受け取りたい | グラフを見る手間をかけたくない |
| 参考価格と実売価格の関係を確認したい | 中古・出品者分析まで深掘りしない |

## 使い方の基本

### ブラウザ拡張機能を入れる

いちばん手軽なのは、Chromeなどのブラウザに**Keepa拡張機能**を入れる方法です。

1. Chrome ウェブストアで「Keepa - Amazon Price Tracker」を検索してインストール
2. [Keepa公式サイト](https://keepa.com/#!language/5)でアカウント登録（トラッキング通知を使う場合）
3. Amazon.co.jpの商品ページを開く
4. 商品画像の下あたりに**価格推移グラフ**が表示される

拡張機能を入れるだけでも、商品ページを開くたびに過去の価格が確認できます。通知を使わない場合でも、グラフ表示だけならこの段階で十分なことが多いです。

### keepa.comでASINを調べる

拡張機能を使わない場合は、公式サイトで商品を検索します。

1. [keepa.com](https://keepa.com/#!language/5)を開く
2. 画面上部の国を**日本**に設定
3. 商品名、またはASIN（例: `B0CSKFY1Q2`）を検索
4. 商品ページでグラフと価格表を確認

Be Primeの記事に載っているASINをそのまま検索すると、同じ商品の推移をすぐ見られます。

### トラッキング（値下げ通知）を設定する

「この価格以下になったら知りたい」場合は、Keepaの**トラッキング**機能を使います。

1. Keepaに無料アカウントでログイン
2. 対象商品のページを開く
3. 希望価格や通知条件を設定してトラッキングに追加
4. 条件を満たしたときにメール等で通知

Prime Dayやタイムセールの前に候補商品を登録しておくと、値下げのタイミングを逃しにくくなります。通知の上限や詳細条件は、無料版と有料版で異なるため、設定画面の表示を確認してください。

## グラフの見方

Keepaのグラフは色分けされています。買い物目的なら、まず次の線を押さえれば十分です。

| 色・項目 | 意味 | 買い物での見方 |
|----------|------|----------------|
| オレンジ | Amazon.co.jpの販売価格 | **いちばん重要**。いまのセール価格が過去のどの位置かを見る |
| 青 | マーケットプレイス（新品） | 第三者出品の値動き。Amazon直販と比較するときに使う |
| 黒 | 中古 | 中古を買う場合の参考 |
| ピンク | タイムセール（Lightning Deal） | 短期セールがいつあったかを確認 |
| 緑 | 売れ筋ランキング（BSR） | 有料機能。人気の波を見たいとき |

### セール価格が「本当にお得か」を見るコツ

グラフで次の3点を確認すると、判断しやすくなります。

- **直近3〜6か月の最安値** — いまの価格がその付近か、まだ高い位置にあるか
- **Prime Day・大型セール前後の谷** — 毎年同時期に下がる商品かどうか
- **参考価格表示との関係** — Amazonの「参考価格」が、Keepa上の実売推移と大きくずれていないか

例えば「参考価格<strong>36,000円</strong>から<strong>20%OFF</strong>」と表示されていても、グラフ上で過去3か月の実売が<strong>28,000円台</strong>まで下がっていたなら、**いまは平均的なセール水準**と捉えた方がよい場合があります。逆に、半年以上下がっていない価格帯なら、買い時としては前向きに検討しやすいです。

SNSや他サイトの「安い！」といった声だけを鵜呑みにせず、商品ページの現在価格と参考価格をその都度確認することをおすすめしています。当サイトでもなるべく正確な情報を掲載するように努めますが、情報が古くなっている場合もあります。Keepaはその確認作業を補助するツールです。

## 無料版と有料版の違い

Keepaは無料でも価格グラフの確認ができますが、**表示期間や分析機能に差**があります。料金は時期やプランで変わるため、最新は[Keepa公式の料金ページ](https://keepa.com/#!language/5)で確認してください。

| 機能 | 無料版 | 有料版（目安） |

**一般の買い物**なら、無料版＋拡張機能で「過去の値下げタイミングを見る」用途はほぼ足ります。**本格的な商品リサーチや物販**をする場合は、有料プランの検討余地があります。

## セール時期との組み合わせ方

Keepaは、Be Primeで紹介しているようなセール記事と併用すると効果的です。

| シーン | Keepaの使い方 |
|--------|---------------|
| タイムセール | 表示価格が直近最安かグラフで確認 → [タイムセールの見方](/posts/amazon-time-sale-guide/)と併用 |
| Prime Day | 候補商品を事前にトラッキング登録 → [Prime Day攻略ガイド](/posts/prime-day-2026-guide/)と併用 |
| 欲しい商品が定番価格 | 過去のセール時の底値を確認し、次の大型セールまで待つ判断材料にする |
| 記事で気になった商品 | 記事のASINをKeepaで検索し、執筆時点よりさらに安くなっていないか確認 |

Keepaは**買うかどうかの最終判断ツール**ではなく、Amazon.co.jpの商品ページ表示を補う参考情報です。在庫・送料・クーポン・ポイントは、必ずAmazon側の表示で確認してください。

## よくある質問

### Keepaは安全に使える？

Keepaは長年使われている価格トラッキングサービスで、ブラウザ拡張機能もChromeウェブストア等から配布されています。ただし拡張機能は権限を求められるため、**公式ストア経由でインストール**し、不要になったら削除する運用が無難です。

### Amazon公式の「価格の推移」と何が違う？

Amazon商品ページにも価格推移を表示する場合がありますが、Keepaは**より長い期間・多くの価格種別（Amazon直販、マーケットプレイス、セール時など）**を1つのグラフで見られる点が強みです。両方を見比べても問題ありません。

### スマホでも使える？

Keepaのスマホアプリ（iOS / Android）があります。外出先でバーコードスキャンや価格確認をしたい場合に便利です。日常のセールチェックは、PCブラウザ＋拡張機能の組み合わせが手軽なことが多いです。

### 無料版だけで買い時判断はできる？

多くの場合は可能です。過去数か月のAmazon直販価格（オレンジの線）が見られれば、「いま買う／待つ」の大まかな判断はつけられます。細かいランキング分析や大量の商品を一括リサーチする必要がなければ、有料版は必須ではありません。

### Keepaに表示が出ないときは？

対象商品が新規出品で履歴が少ない、一時的にデータ取得できない、拡張機能が無効になっている、などの理由が考えられます。別ブラウザで試す、Keepa公式サイトでASIN検索する、Amazonページを再読み込みする、の順で確認してみてください。

## まとめ

[Keepa](https://keepa.com/#!language/5)は、Amazonセールの**価格推移をグラフで確認し、買い時を見極める**のに使える定番ツールです。拡張機能を入れて商品ページのグラフを見るだけでも、「参考価格から○%OFF」が本当にお得なのかを判断しやすくなります。

Be Primeではセール商品や攻略記事を随時更新していますが、購入の最終判断は**Amazon.co.jpの商品ページ**を正として行い、必要に応じてKeepaで過去の値動きを確認する流れがおすすめです。
