Amazonでセール情報を調べていると、**ASIN**（エーエスアイエヌ）という英字と数字の並びを見かけることがあります。Be Primeの商品記事でも `B0DL9MHFJJ` のようなコードを載せていますが、これは**Amazon上で商品を一意に指すID**です。型番や商品名だけだと色違い・容量違いを取り違えやすいため、**買う前にASINを確認しておく**と失敗しにくくなります。

## ASINとは

**ASIN**（Amazon Standard Identification Number）は、Amazonが商品を管理するために使う**識別番号**です。Amazon.co.jpの商品ページやURLに表示され、同じ商品かどうかを判別する際の基準になります。

| 項目 | 内容 |
|------|------|
| 正式名称 | Amazon Standard Identification Number |
| 読み方 | エー・エス・アイ・エヌ（アシン） |
| 桁数 | 基本10文字（英数字） |
| 例 | `B0DL9MHFJJ`（UGREEN充電器）、`B0DCFPWDK5`（ハイセンステレビ） |
| 発行元 | Amazon（書籍はISBNをASINとして流用する場合あり） |
| 主な用途 | 商品ページの特定、価格ツールでの検索、アフィリエイトリンク |

ASINは**Amazonの店舗内ID**です。JANコード（バーコード）やメーカー型番とは別物なので、「型番が同じ＝ASINも同じ」とは限りません。

## 型番・JANコードとの違い

セール時に混乱しやすいのが、**商品名・型番・ASINが別々に存在する**点です。

| 種類 | 誰が付ける | 例 | 役割 |
|------|-----------|-----|------|
| ASIN | Amazon | `B0CSKFY1Q2` | Amazon上でその出品ページを特定 |
| 型番 | メーカー | `881046-0010`、`X759` | 製品の型式名 |
| JAN/EAN | 流通標準 | 13桁のバーコード数字 | 店頭・物流での識別 |

たとえばテレビなら、画面サイズや限定カラーごとに**ASINが分かれる**ことがあります。記事やSNSで「安い！」と書かれていても、**別サイズ・別モデルのASIN**を見ている可能性があるため、購入直前に自分が開いているページのASINを確認する習慣が有効です。

## 商品ページでASINを確認する方法

Amazon.co.jpでは、次のいずれかでASINを確認できます。

### URLから読み取る

商品ページのURLに `dp/` または `gp/product/` の直後にASINが入っています。

```
https://www.amazon.co.jp/dp/B0DL9MHFJJ
                              ↑ これがASIN
```

`/dp/` 形式がいちばんよく使われます。短縮URLやアプリから共有されたリンクでも、最終的な商品ページのURLにASINが含まれます。

### 商品情報欄を見る

ページ下部の**商品情報**（商品の仕様・詳細）に **ASIN** の項目があります。スマホアプリでも、商品詳細を展開すると同様の欄で確認できる場合が多いです。

### Be Primeの記事から辿る

Be Primeのセール記事では、対象商品のASINを frontmatter と本文に記載しています。記事内の商品カードや `::product{B0XXXXXXXX}::` からAmazonの商品ページに進めば、**記事で紹介しているのと同じASIN**かどうかを照合できます。

## こんな場面でASINが役立つ

| シーン | ASINの使い方 |
|--------|-------------|
| セール価格の比較 | [Keepaの使い方](/posts/keepa-amazon-price-tracker-guide/)でASIN検索し、過去の値下げを確認 |
| 色・容量の取り違え防止 | 黒と白、50V型と55V型など、バリエーションごとにASINが異なることが多い |
| 記事と商品ページの照合 | ブログの紹介文と、今開いているAmazonページが同一か確認 |
| ほしい物リストの共有 | URLごと渡すより、ASINをメモしておくと商品特定しやすい |
| 旧モデル・新モデルの見分け | 型番が似ていてもASINが違えば別商品ページ |

## バリエーション商品と親ASIN

色やサイズ違いがある商品は、**子ASIN**（各バリエーション）と**親ASIN**（一覧ページ）に分かれることがあります。

- **子ASIN** — 実際にカートに入れる個別の商品（例: 55V型ブラック）
- **親ASIN** — バリエーション選択画面の親ページ

セール記事で紹介しているのは、通常**購入対象の子ASIN**です。親ページから色やサイズを変えるとASINも変わるため、**最終的に選んだバリエーションのASIN**を確認してください。

## よくある質問

### ASINは日本だけの番号？

いいえ。ASINはAmazonの各国ストアで使われますが、**国ごとに同じ商品でもASINが異なる**場合があります。日本のAmazon.co.jpで買うときは、**.co.jpの商品ページに表示されているASIN**を使ってください。

### 書籍のASINは？

書籍では、ISBN（10桁または13桁）をASINとして扱うことが多いです。紙の本・Kindle版でASIN（ISBN）が異なるため、フォーマット違いの買い間違いにも注意が必要です。

### ASINを知れば最安値がわかる？

ASINは**商品を特定するためのID**であり、価格そのものではありません。最安値や値下げタイミングを知りたい場合は、ASINを使って[Keepa](/posts/keepa-amazon-price-tracker-guide/)やAmazonの商品ページで価格を確認します。

### ASINが変わることはある？

同一商品でASINが突然変わることは稀ですが、**別モデルへのリニューアル、限定版の新規出品、カテゴリ再登録**などで新しいASINのページができることはあります。型番・発売年・商品名をあわせて確認すると安心です。

### 10文字以外のASINは？

ほとんどの一般商品は10文字です。書籍のISBNベースなど、見た目が異なるケースがありますが、Amazon.co.jpの商品情報に表示される**ASIN欄の値**をそのまま使えば問題ありません。

## まとめ

**ASIN**は、Amazon.co.jp上で商品ページを一意に指す10文字前後の識別番号です。URLの `/dp/` の直後や商品情報欄で確認でき、型番・JANコードとは別物です。

セール時は、商品名や「○%OFF」だけでなく、**自分が買おうとしているページのASIN**を一度見ておくと、色違い・サイズ違い・別モデルの買い間違いを防ぎやすくなります。Be Primeの記事に載っているASINと、Amazonの商品ページが一致しているかも、購入前の最終チェックに使ってください。

価格や在庫の正確な情報は、常に**Amazon.co.jpの商品ページ表示**を正として判断してください。
